Project Seed
============

A front-end project skeleton.

Usage
-----

*Dependencies: nodejs, npm, [grunt-cli](https://github.com/gruntjs/grunt-cli).*

```
$ git clone https://github.com/simonsinclair/project-seed.git
$ cd project-seed/
$ npm install
```

### Develop

`$ grunt`

`ctrl + c` to stop developing.

### Deploy

`$ grunt deploy`
